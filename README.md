# Student Management System in C

## Description

This project is a Student Management System developed using the C programming language as part of the CodeAlpha C Programming Internship.

The program helps manage student records through a menu-driven interface. It uses structures, arrays, and functions to store and manipulate student information efficiently.

## Features

* Add Student Record
* Display All Students
* Search Student by Roll Number
* Update Student Details
* Delete Student Record
* Calculate Average Marks
* Menu-Driven Interface

## Concepts Used

* Structures
* Arrays
* Functions
* Loops
* Conditional Statements
* Searching and Record Management

## How to Run

1. Compile the program using any C compiler.
2. Run the executable file.
3. Select an option from the menu.
4. Enter student details as required.
5. View, search, update, or delete records.

## Sample Operations

* Add new student records
* Display all student records
* Search a student by roll number
* Update student information
* Delete a student record
* Calculate average marks
